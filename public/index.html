To start new ISMOP experiment execute following command

curl -k -H "Content-Type: application/json" \
-d '{"experiment":{name: "exp name", "profile_ids":[1,2,3], "start_date": "2012-04-23 18:25:43.511", "end_date": "2012-04-23 18:25:43.511"}}' \
-u username:password https://hypgen.moc.ismop.edu.pl/api/experiments
